Пункт 5. Объясните роль добавленного кода JavaScript.
Код, написанный в разделе JavaScript, выполняет следующие операции:

1. Взаимодействие с кнопкой отправки:
На событие клика по кнопке Отправить заказ назначается слушатель.
При нажатии кнопки запускается функция обработки введенных данных.
2. Сброс сообщений об ошибках:
Перед проверкой полей все предыдущие сообщения об ошибках в элементах с классом .error удаляются.
3. Соберите введенные значения:
Значения, введенные из всех полей (имя, телефон, адрес, дата, время, номер, способ оплаты), получаются с помощью document.getElementById().
Текстовые значения очищаются с помощью функции Trim() для удаления ненужных пробелов.
4. Проверка входных данных:
Введенные значения проверяются для каждого поля:

Убедитесь, что важные поля (имя, телефон, адрес и т. д.) не пусты.
Проверьте формат номера телефона с помощью регулярного выражения (RegExp).
Убедитесь, что введенное число положительное.
Обязательно выберите способ оплаты.
Если возникает ошибка, сообщение об ошибке отображается в соответствующем элементе .error для каждого поля.

5. Показать сводку заказа:
Если все поля действительны (isValid = true), сводка заказа генерируется с использованием данных, введенных пользователем.
Раздел сводки заказа (элемент #summary-content) обновляется с использованием текста, отформатированного с учетом введенных значений.
